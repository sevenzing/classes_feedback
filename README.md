    ```bash
    python3 manage.py runserver
    ```

## To start bot:

1. Create `.env` file
    
    ```bash
    mv .env.example .env
    ```

1. Enter bot token in `.env` file 
1. Start using `docker`:
```bash
docker-compose up --build
```
